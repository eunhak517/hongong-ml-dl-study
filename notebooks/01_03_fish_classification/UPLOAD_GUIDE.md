# 업로드 방법

이 압축 파일을 푼 뒤 `01_03_fish_classification` 폴더 전체를
GitHub 저장소의 `notebooks` 폴더 안에 업로드합니다.

최종 구조:

```text
notebooks/
└── 01_03_fish_classification/
    ├── README.md
    └── 01_03_fish_classification.ipynb
```

GitHub 웹에서:

1. 저장소의 `notebooks` 폴더를 엽니다.
2. `Add file` → `Upload files`를 누릅니다.
3. 압축을 푼 `01_03_fish_classification` 폴더를 업로드 영역에 드래그합니다.
4. 커밋 메시지에 아래 문구를 입력합니다.

```text
study: add fish classification practice
```

5. `Commit changes`를 누릅니다.

자동 진행률 스크립트는 하위 폴더를 재귀적으로 탐색하므로
`01_03_fish_classification.ipynb` 파일을 찾아 진행률을 1/28로 반영합니다.
