# 업로드 방법

압축을 푼 뒤 `02_01_train_test_set` 폴더 전체를  
GitHub 저장소의 `notebooks` 폴더 안에 업로드합니다.

최종 구조:

```text
notebooks/
├── 01_03_fish_classification/
│   ├── README.md
│   └── 01_03_fish_classification.ipynb
│
└── 02_01_train_test_set/
    ├── README.md
    └── 02_01_train_test_set.ipynb
```

## GitHub 웹에서 업로드

1. 저장소의 `notebooks` 폴더로 이동합니다.
2. `Add file` → `Upload files`를 누릅니다.
3. 압축을 푼 `02_01_train_test_set` 폴더를 업로드 영역에 드래그합니다.
4. 커밋 메시지를 입력합니다.

```text
study: add train test set practice
```

5. `Commit changes`를 누릅니다.

자동 진행률 스크립트는 하위 폴더의  
`02_01_train_test_set.ipynb`를 찾아 완료된 절로 계산합니다.
